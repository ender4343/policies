2. Область применения

Политика применяется к доступу к Сервису по адресу https://postiz.ender-server.netcraze.link и обработке данных через официальные API TikTok (Login Kit / Content Posting API).

3. Какие данные мы обрабатываем

  • Идентификаторы TikTok: open_id/union_id, имя профиля, аватар (URL).
  • OAuth-данные: access token, refresh token, сроки действия, выданные TikTok.
  • Контент и метаданные публикаций: видео, обложки, подписи, хэштеги, время/статус публикации, ошибки.
  • Технические журналы: IP/агент браузера, отметки времени, коды ошибок (для диагностики).
  • Аналитика аккаунта (если включена): базовые агрегированные метрики (просмотры, лайки, комментарии).

4. Источники данных
